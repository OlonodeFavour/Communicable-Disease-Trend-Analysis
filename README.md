## 📌 Overview
This project presents an interactive **Nigeria Communicable Disease Trends Dashboard** developed using **Power BI** to analyze reported communicable disease cases in Nigeria between **2008 and 2018**.

The dashboard provides insights into disease trends, distribution patterns, annual case volumes, and comparative disease analysis. It is designed to support public health monitoring, disease surveillance, and data-driven decision-making.

---

## ❗ Problem Statement
Monitoring communicable diseases across multiple years can be challenging due to fragmented datasets and lack of centralized visualization systems.

Key challenges include:
- Difficulty tracking disease trends over time  
- Limited visibility into high-burden diseases  
- Poor understanding of disease distribution patterns  
- Challenges identifying outbreak peaks and declining trends  
- Limited tools for interactive public health analysis  

This project addresses these challenges by transforming raw disease data into an interactive and visually accessible dashboard.

## 🔗 Data Source
The dataset used contains reported communicable disease cases in Nigeria from **2008 to 2018**.

### **Dataset Includes**
- Disease names  
- Total reported cases  
- Annual disease records  
- Yearly trends  
- Disease distribution data  

---

## 🧹 Data Cleaning and Preparation
To ensure accurate analysis, the following preparation steps were performed:

- Removed duplicate entries  
- Handled missing values  
- Standardized disease names  
- Corrected inconsistent date formats  
- Converted numeric fields into proper data types  
- Structured data for Power BI analysis and modeling  

---

## 🔍 Exploratory Data Analysis (EDA)
EDA was conducted to uncover trends and insights within the dataset:

- Analyzed disease trends across years  
- Compared total cases by disease type  
- Identified diseases with the highest prevalence  
- Evaluated yearly fluctuations in reported cases  
- Examined contribution share of diseases

## 📈 Results and Findings

### **Key Insights**
- Cholera recorded the highest number of reported cases  
- Measles and meningitis also contributed significantly to total disease burden  
- Disease cases fluctuated across years, showing outbreak and decline periods  
- A few diseases account for the majority of total reported cases  

---

### **Trend Observations**
- Significant spikes in disease cases occurred around specific years  
- Some diseases showed declining trends over time  
- Disease distribution is uneven across reported conditions  

---

### **Public Health Implications**
- High-burden diseases require increased monitoring and intervention  
- Trend analysis can support outbreak preparedness  
- Data-driven planning can improve healthcare resource allocation  

---

## ⚠️ Challenges and Solutions

### **Challenges**
- Inconsistent formatting within raw disease records  
- Managing multiple disease categories in a single dashboard  
- Ensuring readability with large datasets  

### **Solutions**
- Standardized data formats during cleaning  
- Used filters and slicers to simplify navigation  
- Applied clear visual hierarchy and structured layouts

## 💡 Recommendations
Based on the analysis, the following recommendations are suggested:

- Improve healthcare reporting systems  
- Use predictive analytics for outbreak forecasting
